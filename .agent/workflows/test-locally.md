---
description: How to test the static website locally
---

To test your static website locally, follow these steps:

1. Open a terminal (PowerShell or Command Prompt).
2. Navigate to your project directory:
   ```powershell
   cd .\ngakan.github.io
   ```
3. Start a local server using Python:
   ```powershell
   python -m http.server 8000
   ```
4. Open your web browser and visit:
   [http://localhost:8000](http://localhost:8000)

// turbo
5. If you want to use a different port, just change `8000` to your preferred number.
